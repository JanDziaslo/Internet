# Księga Gości

Treść zadania z moodle

Zadanie polega na napisaniu skryptu php z wykorzystaniem technologii Ajax realizującego funkcjonalność księgi gości
z moderacją.

Skrypt powinien umożliwiać:
- [x] Dodawanie wpisów przez użytkowników.
- [x] Zalogowanie się do panelu administratora.
- [x] Skrypt powinien posiadać prosty edytor umożliwiający wstawianie i interpretację BBCode
- [x] Po zalogowaniu do panelu administracyjnego administrator powinien mieć możliwość przejrzenia wpisów do
moderacji oraz zaakceptowania bądź odrzucenia wpisu.
